## 利用方法
・　未経験者同士で各言語ごとにチャットで教えあったり、ダイレクトメッセージを使って任意の相手に直接質問することができます。　<br>
・　個人のメモと掲示板への投稿ができるので、不明点を記録・検索することができ、インプット・アウトプットが簡単にできます。　<br>
